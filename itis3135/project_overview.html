Project Overview – Charlotte Boxing Club Web App

1. Project Overview

  • Purpose: To create an interactive and informative website for the Charlotte Boxing Club, streamlining communications and recruitment.
  • Intended Users: Current UNCC students.
  • Content: Club information, training schedules, leadership bios, equipment resources, and a signup form.

2. Client Information

  • Name: Charlotte Boxing Club
  • Organization: Charlotte Boxing Club
  • Email: CharlotteBoxing@Charlotte.edu
  • Phone: 555-555-5555

3. Wireframe

Homepage wireframe – Layout includes navigation bar, centered logo and title, login button, and hero image with welcome message overlayed.

4. Site Map

Site Map – Six interconnected pages accessible via the navigation bar, each page leads to informational or interactive content.

5. Website Pages

  1. Home: Reloads the homepage. Contains hero image and welcome message.
  2. About: Informative page about club history, mission, and vision.
  3. Schedule: Displays training schedules and event times in an organized format.
  4. Leadership: Lists bios and contact information of club leadership.
  5. Equipment: Provides a list of required equipment with external resource links (e.g., gloves, mouthguard).
  6. Signup: Contains a form for new members to register with validation.
