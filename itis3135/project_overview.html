Project Overview – Charlotte Boxing Club Web App

1. Project Overview

Purpose: To create an interactive and informative website for the Charlotte Boxing Club, streamlining communications and recruitment.

Intended Users: Current UNCC students.

Content: Club information, training schedules, leadership bios, equipment resources, and a signup form.

2. Client Information

Name: Charlotte Boxing Club

Organization: Charlotte Boxing Club

Email: CharlotteBoxing@Charlotte.edu

Phone: 555-555-5555

3. Wireframe

Homepage wireframe – Layout includes navbar, logo, title, login button, and a large hero image with a welcome message.

4. Site Map

Site Map – Six interconnected pages accessible via the navigation bar. Each leads to a relevant section of the site.

5. Website Pages

  • Home: Reloads the homepage. Contains hero image and welcome message.
  • About: Informative page describing the club's mission and background.
  • Schedule: Displays training sessions and event details in a calendar-style layout.
  • Leadership: Shows bios and photos of club leaders with hover interactivity.
  • Equipment: Provides links and information on required boxing gear (gloves, headgear, etc.).
  • Signup: New member registration form with validation and confirmation message.
